# 注意点
- CircleCI の `circle-token` は、プライベートリポジトリのときに必要になる
  - 誘導があるからすぐに取得できる
- GitHub Actions の `WORKFLOWS_NAME` は、config ファイルの一行目に書かれている `name:` の値のこと
  - 半角スペースは `%20` で書かなければいけない
