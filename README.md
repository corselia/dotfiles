# Usage

#### .vimrc
```.vimrc
source ~/dotfiles/.vimrc
```

#### .zshrc
```.zshrc
source ~/dotfiles/.zshrc
```

#### .gitconfig
```.gitconfig
[include]
  path = ~/dotfiles/.gitconfig
```

# Optional Usage

#### .zshrc
```.zshrc
alias vvimrc='vim ~/dotfiles/.vimrc'
alias vzshrc='vim ~/dotfiles/.zshrc'
alias vgitconfig='vim ~/dotfiles/.gitconfig'
```
